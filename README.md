# CME Faculty Program — landing page

Static landing page + application for the CME faculty talent agency concept.
Deployed via GitHub Pages.

- `index.html` — self-contained (CSS inlined, no build step).
- The application form shows a client-side success state by default. To capture
  real submissions, set `FORM_ENDPOINT` in the inline script to a Formspree URL
  (or any JSON POST endpoint).
